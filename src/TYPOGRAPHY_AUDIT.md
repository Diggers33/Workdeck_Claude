## SPACING & BORDERS

### Border Radius
- **Standard:** 6px (rounded or rounded-lg)
- **Small Elements:** 4px (rounded)
- **Pills/Badges:** 9999px (rounded-full)

### Spacing Units
- **Base Unit:** 4px (Tailwind's default)
- **Common Gaps:** 4px (gap-1), 6px (gap-1.5), 8px (gap-2), 12px (gap-3)
- **Padding:** 12px (p-3), 16px (p-4), 24px (p-6)

---

## NOTES

1. **Font Family:** All text uses Inter font (system default if Inter not loaded)
2. **Line Height:** Using Tailwind defaults (leading-tight, leading-none for specific cases)
3. **Letter Spacing:** Only used on uppercase labels (tracking-wide)
4. **Consistent Hover States:** Most interactive elements darken on hover
5. **Accessibility:** Good color contrast ratios maintained throughout
